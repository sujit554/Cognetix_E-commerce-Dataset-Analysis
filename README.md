# Cognetix_E-commerce-Dataset-Analysis
This project is part of the intermediate stage of my Data Analytics internship. The analysis includes revenue calculation, order analysis, monthly growth trends, top products, country-wise performance, and KPI development.  Tools used include Pandas, NumPy, Matplotlib, Seaborn, and Jupyter Notebook for data cleaning, transformation, visualization.

## Objective
Analyze sales data to extract business insights such as revenue trends, top products, country performance, and growth metrics.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Features
- Revenue analysis
- Monthly growth trend (MoM %)
- Top product analysis
- Country-wise comparison
- KPI dashboard logic

## Project Structure
- data/ → raw dataset
- notebooks/ → analysis notebook
- scripts/ → python scripts
- outputs/ → charts & results

## Insights
- Identified revenue-driving products
- Monthly growth patterns
- Market dominance by country
- Order behavior analysis
